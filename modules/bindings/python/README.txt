BioGraph 6 installation quick-start
===================================

This tarball should be installed via pip inside a virtualenv
running python 3.6 or later.

$ virtualenv --python=python3.6 bgenv

$ . bgenv/bin/activate

(bgenv) $ pip install BioGraph-6.*.tgz
...

(bgenv) $ biograph
